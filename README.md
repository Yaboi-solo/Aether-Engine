# Aether-Engine

Yes, its a game engine. In C++.

## What I want Aether to do in the future.

* Render 2d and 3d graphics
  * Be able to render cutscenes
* Play Audio
  * Background music as well as sound effects
* UI and stuff
* Play test games
* Export projects as executables 

## What I might add to Aether afterwards
* In-built code editor
* Game Engine UI
  * A scene hierachy
  * Material editor

But this ~~might~~ **will** change, mostly for the better.
Until then, I guess we just wait. (while I code the engine)
